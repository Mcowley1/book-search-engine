# book-search-engine
## Description

Book-Search-Engine is a web app that allows you to log-in and search books in different genres with a google api. save your favorite books.

## Tools
> 1. React
> 2. Node.js
> 3. Express.js
> 4. MongoDB and Mongoose
> 5. JWT
> 6. HTML and CSS
> 7. JavaScript

# To Run
> npm start
